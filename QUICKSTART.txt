🎯 START HERE

1. VIEW LOCALLY (Easiest)
   ─────────────────────────
   Option A: Double-click index.html to open in your default browser
   
   Option B: Using Python
     cd /root/projects/southwest-precast/
     python3 -m http.server 8000
     Open: http://localhost:8000

2. NAVIGATE THE SITE
   ─────────────────────────
   Click navigation links to move between pages:
     • Home — Hero, services overview, benefits
     • Services — Detailed service descriptions
     • Gallery — Project images
     • About — Company story and values
     • Contact — Contact form and info
   
   On mobile: Tap hamburger menu (☰) to see navigation

3. TEST FUNCTIONALITY
   ─────────────────────────
   ✓ Click CTA buttons (should navigate smoothly)
   ✓ Fill out contact form and submit
   ✓ Resize browser to test responsive design
   ✓ Hover over cards to see animations
   ✓ Check gallery images on mobile

🚀 TO DEPLOY TO PRODUCTION

1. Choose your hosting (e.g., Netlify, Vercel, GoDaddy, etc.)

2. Upload these 3 files:
   • index.html
   • styles.css
   • script.js

3. Set index.html as your default/home file

4. That's it! No backend needed.
   All functionality is client-side.

5. Domain: Point your domain to the hosting service

❓ TROUBLESHOOTING

Images not loading?
  → Check image URLs are correct and accessible
  → Test in different browser

Colors look wrong?
  → Clear browser cache (Ctrl+F5 or Cmd+Shift+R)
  → Check CSS file is being loaded

Navigation not working?
  → Refresh the page
  → Check JavaScript is enabled in browser
  → Check script.js file exists

Form not submitting?
  → Fill all required fields (marked with *)
  → Check email is valid format
  → Browser console may show errors

For more help, see README.md and SUMMARY.txt
