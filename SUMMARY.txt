🔧 CUSTOMIZATION GUIDE

To modify company information:
  1. Open index.html in a text editor
  2. Search for "+61 430 343 388" or other company details
  3. Replace with your information
  4. Save and refresh in browser

To change colors:
  1. Open styles.css
  2. Find :root { } section at the top
  3. Update --dark-bg, --accent, etc.
  4. All colors throughout site will update automatically

To update gallery images:
  1. Open index.html
  2. Find the gallery section (id="gallery")
  3. Replace image URLs with your own
  4. Update alt text for accessibility

To add more services:
  1. Duplicate a service card in Services page
  2. Update icon, title, description, features
  3. Save and refresh

To modify text content:
  1. Edit directly in index.html
  2. Keep HTML structure intact
  3. Test responsive design after changes
